## 2. node 버전 세팅

```bash
cd frontend

nvm install 22.20.0

nvm use 22.20.0
```

## 3. pnpm 설치

```bash
npm install -g pnpm

```

## 4. 패키지 설치

```bash
pnpm install
```

## 5. 개발 서버 실행

```bash
pnpm dev
```
